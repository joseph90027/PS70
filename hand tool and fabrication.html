[ Theo Jansen Machine ]

Kinetic sculptor Theo Jansen is well-known for creating bionic mobile robotics. His work simulates the walking motion inspired by nature, using a simple rotary input to generate a smooth walking motion.





Taking inspiration from his work, I tried to create a deformable structure using the same principle. There are eleven bars, eight joints in total; among them, two joints are fixed anchor points. Also, I use parametric software grasshopper to simulate the transformable process to have great accuracy in manufacturing.
